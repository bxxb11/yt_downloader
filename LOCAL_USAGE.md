# 本地使用指南

## 📍 这是什么？

这是一个**本地网页应用**，运行在您的电脑上。通过浏览器访问 `http://localhost:5000` 来使用。

## ✅ 特点

- **完全本地化**: 所有数据存储在您的电脑上
- **私密安全**: 不会上传任何数据到云端
- **简单易用**: 通过浏览器界面操作，无需命令行
- **功能完整**: 支持视频信息预览、下载、进度显示、文件管理

## 🚀 如何使用

### 第一次使用

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **启动服务器**
   ```bash
   # Windows
   start_server.bat
   
   # macOS/Linux
   chmod +x start_server.sh
   ./start_server.sh
   
   # 或直接运行
   python app.py
   ```

3. **打开浏览器**
   - 访问: http://localhost:5000
   - 您会看到漂亮的网页界面

4. **开始使用**
   - 粘贴YouTube视频链接
   - 选择视频质量
   - 点击"获取视频信息"
   - 点击"开始下载"
   - 实时查看下载进度

### 日常使用

1. 运行 `start_server.bat` (Windows) 或 `./start_server.sh` (macOS/Linux)
2. 打开浏览器访问 http://localhost:5000
3. 使用完毕后，在终端按 `Ctrl+C` 停止服务器

## 📁 文件存储

- **下载的视频**: 存储在 `downloads/` 目录
- **配置文件**: 项目根目录
- **所有数据**: 都在您的电脑上，不会上传

## 🔒 隐私和安全

- ✅ 所有操作在本地完成
- ✅ 视频文件存储在本地
- ✅ 不会连接到任何外部服务器（除了YouTube）
- ✅ 完全私密，只有您能访问

## ⚙️ 配置

### 修改端口

如果5000端口被占用，可以修改 `app.py`:

```python
app.run(debug=True, host='127.0.0.1', port=8080)  # 改为8080
```

然后访问: http://localhost:8080

### 修改下载目录

修改 `app.py` 中的:

```python
DOWNLOAD_DIR = Path("my_videos")  # 改为您想要的目录
```

## 🛑 停止服务器

在运行服务器的终端窗口中按 `Ctrl+C` 停止服务器。

## ❓ 常见问题

### Q: 为什么访问不了 http://localhost:5000？

A: 检查：
1. 服务器是否正在运行（查看终端窗口）
2. 端口5000是否被其他程序占用
3. 防火墙是否阻止了连接

### Q: 下载的视频在哪里？

A: 在项目目录下的 `downloads/` 文件夹中。

### Q: 可以同时下载多个视频吗？

A: 可以，但建议一次下载一个，避免占用过多资源。

### Q: 关闭浏览器后，下载会继续吗？

A: 会继续，但您需要保持服务器运行（终端窗口打开）。

### Q: 可以在其他设备上访问吗？

A: 可以，但需要修改 `app.py`:
```python
app.run(debug=True, host='0.0.0.0', port=5000)  # 允许局域网访问
```
然后通过 `http://您的IP地址:5000` 访问。

## 💡 使用技巧

1. **书签**: 将 http://localhost:5000 添加到浏览器书签
2. **快捷方式**: 创建 `start_server.bat` 的桌面快捷方式
3. **开机启动**: 可以将启动脚本添加到开机启动项（可选）
